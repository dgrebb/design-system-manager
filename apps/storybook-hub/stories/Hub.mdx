## How Composition Works

Storybook Composition allows you to browse multiple Storybook instances from a single hub. Each connected Storybook appears in the sidebar and loads its stories dynamically.

### Starting the Hub

```bash
# Start the main theme-lab storybook first
bun run dev

# In another terminal, start the hub
bun run dev:hub
```

### Adding a New Storybook Ref

Edit `.storybook/main.ts` to add new refs:

```typescript
refs: {
  'theme-lab': {
    title: 'Theme Lab',
    url: 'http://localhost:6006',
  },
  'new-adapter': {
    title: 'New Adapter',
    url: 'http://localhost:6007',
  },
},
```
